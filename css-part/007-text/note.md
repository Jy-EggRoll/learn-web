# 文本属性

## text-align

设置水平对齐方式。

1. left：左对齐。
2. right：右对齐。
3. center：居中对齐。

```css
.text {
  text-align: left; /* default */
}
```

## text-decoration

设置文本装饰。

1. underline：下划线。
2. overline：上划线。
3. line-through：删除线。

## text-transfrom

设置文本大小写。

1. capitalize：首字母大写。
2. uppercase：全部大写。
3. lowercase：全部小写。

## text-indent

设置文本缩进。

1. 正值：向右缩进。
2. 负值：向左缩进。
