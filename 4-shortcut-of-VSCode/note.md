# VSCode开发者工具快捷键

创建一个html文件后，输入`!`再按下回车，可以快速填充好基本内容。

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```

把一行的内容上移或者下移：`Alt + 上/下`

查找：`Ctrl + F`

替换：`Ctrl + H`
